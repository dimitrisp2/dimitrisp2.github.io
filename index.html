About Me

Get to know the developer behind the code

I'm a passionate full-stack web developer with a deep expertise in PHP and MySQL backend architecture combined with modern JavaScript frontend experiences. With hands-on experience building production applications, I specialize in creating robust, scalable solutions that solve real-world problems.

My approach to development is what I call "co-vibe-coding" — a philosophy that embraces clean, intuitive code with an aesthetic sensibility, mixing AI (LLMs) and real-world experience. It's about writing code & prompting that not only work flawlessly but feel right. Every project is an opportunity to explore creative solutions and push the boundaries of what's possible.

I'm an eternal learner who constantly explores new technologies, frameworks, and design patterns. My GitHub is a living portfolio of experiments, side projects, and continuous skill refinement. I believe the best developers are curious, adaptable, and never stop growing.

✨ My Vibe-Coding Philosophy

No "Please build this" and praying that it will work. AI can be a very good tool for bootstraping (or even fully building) apps & sites fast. Creating beautiful prompts and asking for precise code edits and additions can be better than hand-writing everything.
